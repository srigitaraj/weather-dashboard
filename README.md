🌤 Weather Dashboard (React Project)

This is a modern React-based Weather Dashboard that displays temperature based on the selected date. The project uses month-wise temperature logic to simulate seasonal variations and features a smooth animated gradient background for a professional UI experience.

The application demonstrates React state management, conditional rendering, modular component structure, and responsive design principles.

🚀 Features

Date-based weather prediction

Month-wise temperature logic

Animated gradient background

Glassmorphism UI design

Fully responsive layout
